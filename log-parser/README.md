# Log Parser

A Python tool that analyzes log files and reports error statistics.

## What it does
- Reads a log file and counts total lines
- Categorizes log levels (INFO, ERROR, WARNING)
- Tracks unique log messages using a set
- Handles missing files gracefully

## How to run
python3 log_parser.py

## Skills used
Python, File I/O, Dictionaries, Sets, Error Handling
